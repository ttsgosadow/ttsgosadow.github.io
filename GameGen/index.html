Types of games to generate

When playing Malifaux I usually use the latest ruleset. Currently that is Gaining Grounds. So initially this tool only generated strategies and schemes from that pool. My intention is to make this tool as broad as possible and encompass all available strat/scheme pools, such as:

  • Strategies and schemes from the base book (book 1)
  • Strategies and schemes from the Gaining Grounds package (GG16, '17, etc)
  • Special strategies and schemes such as the story encounters, faction specific schemes, etc. This one is low on my to-do list, though

About

I don't have much time to play the game. So when I get to play, I want to have as much time for playing the game as possible. Generating a game can take quite some time, so I thought of this game generator. It's currently generating GG16, but will add other sets when they become available. I use this mainly to generate a game and then send the perma-link to my opponent. Hope it's useful for others as well.
